h3Sarmatia is a town-creatures based template for random map generator of Heroes of Might and Magic 3 Horn of the Abyss HDMod

To install it, download the .zip archive and unpack it to 

Heroes3/HotA_RMGTemplates/h3Sarmatia

Make sure, that following file is present: 

HotA_RMGTemplates/h3Sarmatia/rmg.txt
